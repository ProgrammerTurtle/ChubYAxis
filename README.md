# ChubYAxis
ChubY is a custom **high-performance** Y axis assembly for my 2018 Ender 3. It is called ChubY due to it being very **chunky**.

ChubY Features:
- 4040 Base
- 3090 Y Extrusion
- Dual MGN12
- Huge Braces
- And Cool Looks !
